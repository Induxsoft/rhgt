# Rhgt (Tesorería de recursos humanos).
El paquete Rhgt (Tesorería de recursos humanos) de V12 es el módulo que permite administrar la tesorería vinculada a los recursos humanos, integrando configuraciones de categorías, cuentas y procesos de reembolso de gastos de empleados. Su propósito es asegurar un control financiero claro y ordenado sobre las operaciones relacionadas con el personal, garantizando trazabilidad y cumplimiento.
